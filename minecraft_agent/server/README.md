# Local Minecraft Server

This folder is for a local Java Edition server that the bot can connect to.

## Planned Setup

- Server version: `1.12.2`
- Auth mode: `offline`
- Port: `25565`

## Files

- `server.jar`: vanilla server jar
- `eula.txt`: generated after first run, then set to `eula=true`
- `server.properties`: local bot-friendly defaults
- `launch_server.ps1`: launch script
